How I Started Coding With Zero Experience

Published July 2025 by Decent Tengesu

Before I ever typed a line of code, I thought programming was only for geniuses. But all that changed the day I built my first simple webpage using just HTML and CSS.

My coding journey didn’t begin in a classroom — it started on YouTube, free coding platforms, and hours of trial and error. Now, I build real applications and host my own websites.

Tools That Helped Me Start

  • 💻 Visual Studio Code – my daily code editor
  • 🌐 W3Schools and FreeCodeCamp – for tutorials
  • 📱 Netlify – for launching my websites online
  • 🐙 GitHub – for version control and portfolios

One project at a time, I improved. My proudest moment so far has been building and deploying tech-tengesu-site.netlify.app.

Advice to Beginners Like Me

  1. ✅ Build messy projects — perfection comes later
  2. ✅ Practice more than you watch
  3. ✅ Learn one tool at a time

If I could start from zero and get here, so can you. This is only the beginning of my tech journey — and yours too.
